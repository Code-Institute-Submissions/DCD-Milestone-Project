Machine Learning: Classification

Overview:

Classification identifies to which of a set of categories a new observation belongs. Classification problem is an example of pattern recognition that predict a probability of a given example belonging to each output class. The output variables are often
called labels or categories.

Real life example:

  • Email spam detector
  • Handwritten Digit Recognition.
  • Speech Recognition.
